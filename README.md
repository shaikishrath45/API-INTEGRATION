# 🌍 Country Explorer – Public API Integration Project

## Project Overview

Country Explorer is a responsive web application developed using HTML, CSS, and JavaScript that allows users to search for countries and view important information about them. The project demonstrates the implementation of Public API integration in modern web development by retrieving country-related data dynamically and displaying it through an interactive user interface.

The primary objective of this project is to understand how web applications communicate with external services using APIs and how dynamic content can be displayed without refreshing the webpage. Users simply enter the name of a country into the search field, and the application fetches relevant information and presents it in an organized format. This project provides practical experience in API integration, JavaScript programming, data handling, and frontend development.

## Features

The Country Explorer application includes the following features:

* Search functionality to find information about a specific country.
* Dynamic retrieval of country data using a Public API.
* Display of country details such as:

  * Country Name
  * Capital City
  * Currency
  * Population
  * Region
  * Languages
  * Time Zone
  * National Flag
* Responsive and user-friendly interface.
* Dynamic content updates without page refresh.
* Error handling for invalid country searches.

## Working Process

The application follows a simple workflow:

1. The user enters the name of a country in the search box.
2. When the Search button is clicked, JavaScript captures the input value.
3. A request is sent to a Public API to retrieve country information.
4. The received data is processed and matched with the user's input.
5. Relevant details such as capital city and additional country information are extracted.
6. The information is displayed dynamically on the webpage.
7. If the country is unavailable or invalid, an error message is shown.

This process demonstrates how web applications can interact with external APIs and present data efficiently to users.
